# Documentação de Bibliotecas do Frontend

Este registro detalha as dependências do projeto, suas versões fixas e os links para consulta rápida à documentação oficial.

## Core e Build Tool

| Biblioteca | Versão | Propósito |
| :--- | :--- | :--- |
| [**React**](https://react.dev/) | 18.2.0 | Biblioteca base para construção da interface. |
| [**Vite**](https://vitejs.dev/) | 8.0.10 | Bundler e ambiente de desenvolvimento. |

## Estilização e UI

* [**Tailwind CSS (4.2.4)**](https://tailwindcss.com/): Framework utilitário para estilização consistente e acelerada.
* [**Lucide React (1.14.0)**](https://lucide.dev/): Biblioteca de ícones leves e customizáveis.
* [**React Hot Toast (2.6.0)**](https://react-hot-toast.com/): Notificações leves para feedback de ações do usuário.
* [**clsx (2.1.1)**](https://github.com/lukeed/clsx): Utilitário para gerenciar classes condicionais de forma limpa.

## Navegação e Consumo de API

* [**React Router Dom (6.30.3)**](https://reactrouter.com/): Gerenciamento de rotas e navegação em Single Page Application.
* [**Axios (1.16.0)**](https://axios-http.com/): Cliente HTTP para comunicação com a API e interceptação de requisições.
* [**TanStack React Query (4.36.0)**](https://tanstack.com/query/v4): Gerenciamento de estado assíncrono, cache e sincronização de dados.

## Gerenciamento de Estado

* [**Zustand (5.0.13)**](https://docs.pmnd.rs/zustand/): Store de estado global minimalista, usada para autenticação e dados compartilhados.

## Formulários e Validação

* [**React Hook Form (7.75.0)**](https://react-hook-form.com/): Performance e flexibilidade na criação de formulários.
* [**Zod (4.4.3)**](https://zod.dev/): Validação de esquemas com foco em segurança de tipos (TypeScript).
* [**@hookform/resolvers (5.2.2)**](https://github.com/react-hook-form/resolvers): Ponte de integração entre o Zod e o React Hook Form.

## Visualização de Dados

* [**Recharts (3.8.1)**](https://recharts.org/): Biblioteca de gráficos baseada em componentes React.

---

**Nota:** Mantenha as versões fixas conforme listado para evitar incompatibilidades. Para novas instalações ou atualizações, consulte a equipe de arquitetura.
